# @spraoi/babel-preset

> A preconfigured, extensible
> [babel preset](https://babeljs.io/docs/en/presets).

## Installation

```bash
yarn add \
  @babel/core \
  @spraoi/babel-preset \
  babel-plugin-styled-components \
  babel-preset-gatsby
```

## Usage

Add the following configuration to the root of your project:

**.babelrc**

```json
{
  "presets": ["@spraoi/babel-preset"]
}
```
